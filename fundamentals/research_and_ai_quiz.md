## Answer Key (Teacher Copy)

1. **B** — When AI generates false information that sounds confident and correct. AI can cite fake studies, invent statistics, and state false facts with total confidence.

2. **B** — `site:reddit.com learn JavaScript 2026` uses the site: operator to limit results to Reddit and includes relevant keywords.

3. **B** — Search for the same claim in other credible sources. This is the "F" in SIFT (Find better coverage). If only one site makes this claim, it's likely exaggerated or false.

4. **Learning:** "I'm struggling to understand CSS Grid. Can you explain how grid-template-columns works with a simple example?" — you're asking AI to help you understand so you can use the knowledge yourself. **Cheating:** "Write my CSS Grid assignment for me" — you're submitting AI's work as your own and learning nothing.

5. Good examples: "Give me 5 practice questions about [topic] and let me try to answer before showing solutions." / "Explain [concept] like I'm in middle school, then ask me a follow-up question to check if I understood." / "I think [X] works like [Y] — am I right? If not, explain where my understanding is wrong."
